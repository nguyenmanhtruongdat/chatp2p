
# Chat Peer to Peer App  




## Features

- Chat beetween 2 peer (client) using Java


## ✅ Installation

To run the Server: ``` java –jar .\jar\chatServer_1.0.jar portNumber ```

To run the Client: ``` java -jar .\jar\chatClient_1.0.jar ipAddress portNumber ```
